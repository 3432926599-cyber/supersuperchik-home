## ✦ About

Personal portfolio & brand homepage built with the **Dark Terminal** aesthetic. A single-page showcase of projects, services, skills, and contact — designed to feel like a polished terminal interface rather than a generic portfolio template.

### Design Philosophy

- **Dark Terminal** — `#0d0d0d` background with scanline texture, green/amber accent system
- **Typography-first** — Syne display font + JetBrains Mono for code-like precision
- **WeChat X5 Ready** — Full compatibility with WeChat's built-in browser (X5 kernel)
- **Zero AI Plastic** — No purple gradients, no glassmorphism, no Inter/Roboto

| Layer | Technology |
|-------|-------------|

| **Deploy** | Cloudflare Pages via Wrangler |
